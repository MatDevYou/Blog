Elementi e tag


Un tag è una keyword del linguaggio racchiusa tra parentesi angolari (<>).
Esempi di tag sono <'html'>, <'body'> e <'h1'>. I tag HTML non sono "case sensitive" ciò significa che scrivere <'head> o <'HEAD> è esattamente la stessa cosa.
In ogni caso la consuetudine è quella di scrivere i tag in minuscolo.

Elementi contenitori e autonomi


Molti elementi in HTML servono per descrivere porzioni di pagina, aree, o contenuti. Pertanto un elemento HTML è quasi sempre un contenitore e il suo contenuto è delimitato da:

@ tag di apertura
@ tag di chiusura 

                    <!doctype html>
                    <html lang="it">
                        <head>
                            <title>Ciao Mondo!</title>
                        </head>
                        <body>
                            <h1>Ciao Mondo!</h1>
                            <p>Questa è la nostra prima pagina HTML!</p>
                            <img src="immagine.jpg">
                        </body>
                        </html>

Attributi


I tag HTML possono essere corredati di uno o più attributi, che servono per meglio specificare la funzione o la tipologia dell'elemento, per memorizzare dati o per arricchire di significato il contenuto.
Un tag con attributi si scrive in questo modo: 

<input type="email" name="email" placeholder="Scrivi il tuo indirizzo email...">


Il tag input indica genericamente un elemento che consente agli utenti di inserire delle informazioni.
Grazie agli attributi però, possiamo specificare che vogliamo un indirizzo email (attributo type) e possiamo comunicarlo all'utente con un messaggio (attributo placeholder).
